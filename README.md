##### 个人博客前台

###### 前端

###### vue3 + vite + element-plus + axios + vue-router + pinia + sass

###### 后端

###### java + SpringBoot + MyBatis + JWT + Redis + mysql

###### Build Setup

```bash
# clone the project
git clone https://github.com/Hedongliang0518/blog_user.git

# enter the project directory
cd blog_user

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:8888
